# X-ui (Modernized & Localized)

Панель управления для Xray с поддержкой множества протоколов и пользователей.
**Модернизированная версия с полной локализацией (Русский, English, Chinese), усиленной безопасностью и оптимизированным кодом.**

## Функциональные возможности

- **Системный мониторинг**: Статус сервера, использование ресурсов.
- **Многопользовательский режим**: Поддержка множества пользователей и протоколов с визуальным управлением.
- **Поддерживаемые протоколы**: VMess, VLESS, Trojan, Shadowsocks, Dokodemo-door, Socks, HTTP.
- **Расширенная конфигурация**: Настройка транспорта, TLS, XTLS и других параметров.
- **Управление трафиком**: Статистика, лимиты трафика, установка срока действия аккаунтов.
- **Шаблоны конфигурации**: Возможность настройки шаблонов Xray.
- **HTTPS доступ**: Поддержка доступа к панели через HTTPS (свой домен + SSL).
- **Локализация**: Полная поддержка Русского, Английского и Китайского языков.
- **Telegram бот**: Уведомления о трафике, входе в панель и истечении срока действия.
- **Безопасность**: Хеширование паролей (bcrypt), настройки приватности для уведомлений.

## Установка и Обновление

### Автоматическая установка (Linux)

```bash
bash <(curl -Ls https://raw.githubusercontent.com/Ivormise/x-ui_ru/main/install.sh)
```
*Примечание: Скрипт установки поддерживает выбор языка.*

### Удаление

```bash
systemctl stop x-ui
systemctl disable x-ui
rm /etc/systemd/system/x-ui.service -f
systemctl daemon-reload
systemctl reset-failed
rm /etc/x-ui/ -rf
rm /usr/local/x-ui/ -rf
rm /usr/bin/x-ui -f
```

## Telegram Бот

Для настройки уведомлений выполните следующие шаги:

1.  **Создайте бота**: Найдите [@BotFather](https://t.me/BotFather) в Telegram, создайте нового бота и получите **API Token**.
2.  **Получите Chat ID**: Найдите [@userinfobot](https://t.me/userinfobot) или аналогичного бота, чтобы узнать свой **Chat ID**.
3.  **Настройте панель**:
    - Перейдите в раздел "Настройки панели".
    - Введите **API Token** и **Chat ID**.
    - Установите расписание уведомлений (например, `@daily` или `0 30 8 * * *` для 8:30 утра).
4.  **Приватность**: В настройках можно отключить отображение IP-адреса и имени хоста в уведомлениях для повышения безопасности.

## Безопасность и Производительность

В данной версии внесены важные улучшения:
- **Хеширование паролей**: Пароли больше не хранятся в открытом виде. Используется `bcrypt` с автоматической миграцией при первом входе.
- **Кэширование IP**: Оптимизировано определение IP-адреса сервера для снижения нагрузки.
- **Оптимизация БД**: Уменьшено количество записей в базу данных при обновлении статистики трафика.

## Системные требования

- CentOS 7+, Ubuntu 16+, Debian 8+
- **Go 1.22+** (для сборки из исходников)

## Миграция с v2-ui

```bash
x-ui v2-ui
```
*После миграции остановите v2-ui и перезапустите x-ui.*
