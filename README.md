# X-ui (Modernized & Localized)

Панель управления для Xray с поддержкой множества протоколов и пользователей.
**Модернизированная версия с полной локализацией (Русский, English, Chinese) и оптимизированным кодом.**

## Функциональные возможности

- **Системный мониторинг**: Статус сервера, использование ресурсов.
- **Многопользовательский режим**: Поддержка множества пользователей и протоколов с визуальным управлением.
- **Поддерживаемые протоколы**: VMess, VLESS, Trojan, Shadowsocks, Dokodemo-door, Socks, HTTP.
- **Расширенная конфигурация**: Настройка транспорта, TLS, XTLS и других параметров.
- **Управление трафиком**: Статистика, лимиты трафика, установка срока действия аккаунтов.
- **Шаблоны конфигурации**: Возможность настройки шаблонов Xray.
- **HTTPS доступ**: Поддержка доступа к панели через HTTPS (свой домен + SSL).
- **Локализация**: Полная поддержка Русского, Английского и Китайского языков.
- **Telegram бот**: Уведомления о трафике и входе в панель.

## Установка и Обновление

### Автоматическая установка (Linux)

```bash
bash <(curl -Ls https://raw.githubusercontent.com/Ivormise/x-ui_ru/main/install.sh)
```
*Примечание: Скрипт установки теперь поддерживает выбор языка.*

### Удаление

Для полного удаления панели и очистки системы выполните:

```bash
systemctl stop x-ui
systemctl disable x-ui
rm /etc/systemd/system/x-ui.service -f
systemctl daemon-reload
systemctl reset-failed
rm /etc/x-ui/ -rf
rm /usr/local/x-ui/ -rf
rm /usr/bin/x-ui -f
```

### Ручная установка / Сборка из исходников

Для сборки требуется **Go 1.22** или выше.

1.  Скачайте исходный код.
2.  Перейдите в директорию проекта.
3.  Запустите сборку:

```bash
go mod tidy
go build -o x-ui main.go
```

## Использование Docker

> Образ Docker поддерживается сообществом.

1.  Установите Docker:
    ```bash
    curl -fsSL https://get.docker.com | sh
    ```

2.  Запустите X-ui:
    ```bash
    mkdir x-ui && cd x-ui
    docker run -itd --network=host \
        -v $PWD/db/:/etc/x-ui/ \
        -v $PWD/cert/:/root/cert/ \
        --name x-ui --restart=unless-stopped \
        enwaiax/x-ui:latest
    ```

## SSL Сертификаты

Скрипт поддерживает автоматическое получение SSL сертификатов через Let's Encrypt (DNS API).
Требуется:
- Email, зарегистрированный в Cloudflare.
- Cloudflare Global API Key.
- Домен, привязанный к Cloudflare.

Сертификаты сохраняются в `/root/cert`.

## Telegram Бот

Настройка через веб-интерфейс панели:
- Токен бота.
- Chat ID (куда отправлять уведомления).
- Расписание уведомлений (Crontab синтаксис).

Функции бота:
- Уведомления о трафике.
- Оповещения о входе в панель.
- Напоминания об истечении срока действия.

## Системные требования

- CentOS 7+
- Ubuntu 16+
- Debian 8+
- **Go 1.22+** (для сборки из исходников)

## Миграция с v2-ui

Если у вас установлен v2-ui, вы можете импортировать аккаунты:

```bash
x-ui v2-ui
```
*После миграции остановите v2-ui и перезапустите x-ui во избежание конфликтов портов.*
